Automate AWS Resource Provisioning using Python (boto3)

Developed a Python automation tool using boto3 to provision AWS infrastructure programmatically.
Automated creation of IAM users, S3 buckets, and EC2 instances, eliminating manual console operations.
Configured AWS authentication using the AWS CLI and IAM access keys.
Implemented exception handling and logging for reliable infrastructure provisioning.
Verified created resources through the AWS Management Console and AWS APIs.
